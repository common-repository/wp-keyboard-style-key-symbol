== Installation ==
= WP Keyboard Style Key Symbol =
1. Go to Plugins > Add New Plugin*, search for **WP Keyboard Style Key Symbol** and click "*Install now*"
2. Alternative method, download the plugin and Upload the WP Keyboard Style Key Symbol folder to the /wp-content/plugins/directory
3. Activate the plugin
4. Now Go to Plugin settings page under WordPress Settings page.
5. And Choose your Favourite color for the WP Keyboard Style Key Symbol
6. That's all

OR, See a detailed doc, [how to install a WordPress plugin](https://codex.wordpress.org/Managing_Plugins#Installing_Plugins).
